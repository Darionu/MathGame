## MathGame

### Live Demo

* Go to <http://darionu.pl:3000>

### Setting Up

* Clone this repo
* Do `meteor npm install` to install dependencies
* Make sure you've installed Meteor locally

### Running The App

Simply start your app with `meteor`.
Then you can access the app on <http://localhost:3000>

> ```js
> rm -rf .meteor/local
> rm -rf node_modules
